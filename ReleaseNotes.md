ManpWIN – Latest Development Build

This release focuses on stability improvements and bug fixes.

New Features:
- Hailstone (Collatz) sequence visualisation
- SVG export for Hailstone (experimental)

Fixes:
- Resolved resize lockups in non-raster fractals
- Improved handling of rendering modes (CurrentRenderMode issues)

Notes:
- Hailstone help documentation is not yet integrated into the main help system.
- SVG output may differ slightly from on-screen colours.
