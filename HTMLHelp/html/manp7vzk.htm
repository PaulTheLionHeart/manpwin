File Menu

File Operations

GOpen PNG image file
OOpen parameter file
Parameter files store favourite locations and settings for fractals. Note that FRACTINT parameter files use a different format and are loaded via the Fractal Type menu.
KOpen Kalles Fraktaler parameter file
Loads a subset of parameters supported by Kalles Fraktaler.
IOpen animation script file
Animation frames are generated using a script file that controls the sequence.
Shift+LOpen animation frame PNG list (*.LST)
Loads a list of PNG frames previously saved by ManpWIN and generates an MPEG-2 animation.
SSave parameter file
ASave image (PNG)
Ctrl+Shift+SSave both parameters and image
Ctrl+KSave Kalles Fraktaler parameter file
Save SVG vector image
Available only for certain fractals (e.g. Hailstone). Exports a scalable vector version.
Enter / EscExit application
