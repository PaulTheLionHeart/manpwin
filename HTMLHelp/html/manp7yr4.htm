Keyboard Commands

Help:

F1                                Help

 

File Menu:

'O' key                         Open parameter file

'G' key                         Open PNG image file

‘I’ key                         Open animation script file

Shft+L                         Open LST file to load animation frames for building an MPEG file

‘K’ key                        Open Kalles Fraktaler parameter file

'A' key                         Save image file in PNG format

'S' key                          Save parameter file

'Enter' key                   Exit via confirmation dialogue

'Esc' key                      Exit via confirmation dialogue

Shft+Enter                  Exit immediate

 

Edit Menu:

Ctrl+C                         Copy image to clipboard

Ctrl+Z                         Undo

Ctrl+Y                         Redo

 

Fractal Menu:

'T' key                         Select fractal type

Ctrl+T                         Restart fractal with updated parameters

'X' key                         Fractal Options

'D' key                         Set up parameters for 3D transforms

'W' key                        Toggle Perturbation. This is useful for some fractals when the arbitrary precision gets too slow, switches to perturbation

'Tab' key                      Display current location and allow copying data into the clipboard

Ctrl+F                         Enter Fractal Co-ordinates

'F' key                          TieraZon FD filter options (only applies to TieraZon filter types 27, 28, 33 - 38, 48 and 49)

 

Options Menu:

'3' key                          Toggle 3D translation mode

'Space' bar                   Toggle Julia Set and Mandelbrot fractals

Ctrl+Space                  Real time Julia (only works in Mandelbrot fractals)

Shft+Space                 Set real time Julia thumbnail size

Shft+Left Button        Toggle Julia for cursor location

'R' key                         Random Dot Stereogram

Shft+R                        Toggle Random Dot Stereogram Guide Spots

'Z' key                         Stereo Pairs

'Y' key                         Add images (used in Fibonacci spirals)

'E' key                         Toggles between zooming along the edge of a rectangle or from the centre

Shft+Z                         Set image size

'B' key                         Toggle displaying status or cursor location in number space

Ctrl+P                         Toggle between ManpWIN and Fractint Palette

Ctrl+M                        Setup multi-threading parameters for perturbation and slope

Shft+D                        Setup stereo pair

 

Animation Menu:

Ctrl+A                         Setup zoom animation

'J' key                          Setup Julia animation (linear, circle or cardiod)

Shft+F                         Setup Fourier animation

Ctrl+I                          Setup Ikeda animation

Shft +I                         Setup Inversion animation

Ctrl+O                         Setup Chaotic Oscillator or Fractal Map animation (other fractals also rotate. Refer Animation Menu).

'P' key                          Setup animation for a specific parameter between user specified values

Shft+M                       Setup Morphing animation between sets of axes (Oscillators, Fractal Maps, Knots and Curves)

Shft+S                         Stop animation

 

While the animation is running, the following keys are active:

'+' key                          Increase speed of the animation

'-' key                          Decrease speed of the animation

'S' key                          Suspend/resume animation

'R' key                         Reverse direction of animation

'A' key                         Save all frames as individual PNG files

'G' key                         Save frames as an animated GIF file (not recommended for large images or large numbers of frames)

'M' key                        Save frames as an MPEG-2 file

'Esc' key                      Escape out of animation display

Space Bar                    Single step

'S' key twice                Resume animation from single step

 

'Esc' key                      Escape from GIF and MPEG creation. These can take a long time for large numbers of frames

 

Colour Menu:

'Q' key                         Load palette colour map

'L' key                         Load true colour map

'N' key                         Save true colour map

Ctrl+N                         Save palette colour map

'+' or ‘-‘ key                Colour cycle

'C' key                         Change colour scheme

Ctrl+S                         Set “inside” colour

Ctrl+E                         Edit Palette

Shft+C                        TieraZon colour type
