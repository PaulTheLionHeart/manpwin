Options Menu

3D Plot                                               (key 3)

This displays the fractal in 3 dimensions.

Julia Set toggle                                  (key Space bar)

The Julia set is drawn for the seed value at the centre of the current fractal. Not all fractals have Julia set interpretation. The flag is ignored if not relevant.

Real time Julia                                  (Ctrl+Space)

It is also possible to generate a small thumbnail Julia set in real time which follows the mouse cursor as it transverses the Mandelbrot set. In this mode, selecting the Julia set will give a full screen image centred on the current location within the Mandelbrot Set.

Set RT Julia Thumbnail Size            (Shft+Space)

The size of the thumbnail is user selectable.

Random Dots Stereogram                (key R)

Random dot stereograms give the illusion of 3 dimensions. They are viewed by going 'cross eyed' and allowing the images to overlap.

Stereo Pairs                                       (key Z)

This is similar to random dot stereograms but there are two images which are brought together.

Add Spirals                                        (key Y)

Indicate if Fibonacci spirals are to be overwritten or to be added to.

Zoom Box Options                            (key E)

If true, this will allow scanning of the edge of a rectangle following the cursor. If false, the zoom box will centre at the original point selected.

Change image size                             (Shft+Z)

Set the fractal image size to any value;

Start

Restart the image calculation

Setup Folder Path

Sets up paths for specific ManpWIN file types.

Note that the user can now select whether to supply their own directories or to use the default user data area. This overcomes many of the issues caused by UAC (User Access Control) in Vista, Windows 7 and beyond. The actual paths are shown for both the application and the user data.

Display Location                               (key B)

Display in the captions bar the current location of the cursor in number space

Setup Multi-threading                      (Ctrl+M)

Sets the number of threads and the number of columns to be calculated. Note that the total width may be less than the full screen. It is best to choose the number of threads to be an integer divider of screen width.

Delays are added to prevent crashes cause by threads not completing on time. Internal perturbation delay greater than 0 will cause the screen to update continuously.

Setup Stereo Pairs                            (Shft +D)

Sets the size of each image. This depends on screen size in pixels. For larger screens, a larger number is required to allow the eye separation to be comfortable. Setting proportion to zero turns off stereo pairs. Stereo depth can also be set.
