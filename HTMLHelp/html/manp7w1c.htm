Setup Options Menu

This menu provides controls for rendering, display, and configuration.

Keyboard shortcuts are shown in brackets.

3D Plot (3)

Displays the current fractal in three dimensions.

Julia Fractal (Space)

Generates the Julia set using the seed value at the centre of the current fractal. Not all fractals support a Julia interpretation.

Real Time Julia (Ctrl+Space)

Displays a live thumbnail Julia set that follows the mouse cursor as it moves across the Mandelbrot set. Selecting this option generates a full-screen Julia image at the current cursor location.

Set Real Time Julia Thumbnail Size (Shift+Space)

Adjusts the size of the real-time Julia preview window.

Random Dot Stereogram (R)

Creates a stereogram image that produces a 3D effect when viewed using relaxed or cross-eyed viewing techniques.

Toggle Random Dot Stereogram Guide Spots (Shift+R)

Toggles guide markers to assist in viewing stereograms.

Stereo Pairs (Z)

Displays two side-by-side images that can be combined by the viewer to produce a stereoscopic 3D effect.

Add Images (Y)

Adds overlay images such as spirals or guides to the current fractal.

Zoom Along Edge (E)

Controls zoom behaviour. When enabled, the zoom box follows the cursor edge.

Change Image Size (Shift+Z)

Allows the fractal image size to be set to a custom resolution.

Start

Begins or restarts the image calculation.

Setup Folder Paths

Configures file storage locations for ManpWIN data. Users can choose between custom directories or the default user data area, helping to avoid issues with Windows User Account Control (UAC).

Use Default (Fractint) Palette (Ctrl+P)

Switches to the default Fractint colour palette.

Display Location (B)

Displays the current cursor position in the complex plane within the caption bar.

Generate Statistics File for MPEG Generation

Creates a statistics file used for generating MPEG animations.

Setup Multi-threading for Perturbation and Slope (Ctrl+M)

Configures the number of threads used for advanced rendering modes.

Setup Stereo Pair (Shift+D)

Adjusts stereo image size and depth for comfortable viewing.

Toggle Display Palette (Shift+T)

Shows or hides the current colour palette display.

Note: Some options may not apply to all fractal types and may behave differently depending on the rendering mode (standard, perturbation, or slope).
