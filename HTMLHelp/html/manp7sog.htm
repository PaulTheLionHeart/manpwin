Colour Cycling

Overview

Colour cycling produces dynamic visual effects by rotating the palette used to display the fractal. This can create the appearance of motion without recalculating the image.

Usage

  • + key - cycle colours forward
  • - key - cycle colours backward

In some cases, the key may need to be pressed more than once to begin cycling.

Notes

  • Colour cycling is slower when using true colour palettes due to the larger number of colours involved.
  • Different palette files can produce dramatically different visual effects.
