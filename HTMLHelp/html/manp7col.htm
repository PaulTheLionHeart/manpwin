More About Colour

Overview

ManpWIN supports two primary colour modes: True Colour and Palette (MAP) mode. Each offers different advantages depending on the fractal and rendering method.

True Colour Mode

True colour mode generates colours algorithmically using RGB start values and increments. This allows smooth gradients and is particularly effective for fractals with large iteration counts.

  • Supports smooth colour transitions
  • Ideal for deep zooms and high iteration renders
  • Allows generation of random colour palettes

Palette (MAP) Mode

In MAP mode, colours are defined by a fixed palette file. Each iteration value maps to a specific RGB entry within the palette.

  • Uses FRACTINT-compatible *.map files
  • Provides precise control over colour bands
  • Well suited to classic fractal rendering styles

Switching Modes

Use Ctrl+P to toggle between True Colour and MAP modes.

Note: Random palette generation is only available in True Colour mode.

Tierazon Colouring

Tierazon colouring and filtering use a different colour model from standard ManpWIN modes.

To exit Tierazon colouring:

  • Load a palette map using Q
  • Or load a true colour map using L
  • Alternatively, reselect the fractal using T

Notes

For advanced palette editing, see Palette Editing.

Different fractals and rendering modes may respond differently to colour selection. Experimentation with both modes is encouraged to achieve the desired visual effect.
