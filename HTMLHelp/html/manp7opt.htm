Fractal Options

Overview

The Fractal Options dialog provides detailed control over how fractals are calculated, coloured, and displayed. These settings affect rendering behaviour, colouring methods, and mathematical interpretation.

Access this dialog using the X key.

Basic Parameters

Threshold

Controls bailout sensitivity. Higher values allow more iterations before escape.

Tesseral Fill Colour

Colour used when tesseral rendering is enabled. A value of -1 disables this feature.

Log Value

Applies logarithmic scaling to colouring. A value of 0 disables log scaling.

Biomorph Colour

Highlights points near axes. A value of -1 disables this effect.

Decomposition Colour

Applies colouring based on angular decomposition.

Special Colour

Used by certain fractal types for custom colouring effects.

Orbit Colour

Sets orbit display colour (RGB hex format).

Background Colour

Used for L-systems and IFS fractals.

Radius / Centre

Defines the region of interest for rendering.

Plotting Methods

Determines how the fractal is rendered.

  • Solid Guessing - Fast rendering using interpolation
  • Boundary Tracing - Traces edges of fractal regions
  • Tesseral - Block-based rendering
  • Single / Double Pass - Progressive refinement
  • Horizontal / Vertical Blinds - Strip-based rendering
  • Spiral Inwards - Spiral rendering pattern
  • Slope Forward Differencing - Enables slope shading

Colour Smoothing

Smooths colour transitions between iteration bands.

Transfer Factor

Controls colour scaling intensity.

Light Direction

Defines light angle for slope shading.

Outside Filters

Applied to points outside the fractal set.

  • Real, Imaginary, Sum, Multiply
  • ArcTan
  • Perturbation Filters
  • Epsilon Cross
  • Star Trails
  • Potential Slope
  • TieraZon Filters

No Outside Filter disables all effects.

Inside Filters

Applied to points inside the fractal set.

  • Z Magnitude
  • Beauty of Fractals (various modes)
  • No Inside Filter

Distance Estimation

Distance Estimator

Enables distance-based rendering.

Width Factor

Controls thickness of rendered boundaries.

Bailout Test

Defines how escape conditions are evaluated.

  • MOD
  • Real
  • Imaginary
  • OR / AND
  • Manh / Mann

Coordinate System

Transforms the coordinate space used for rendering.

  • Cartesian
  • Spherical
  • Cylindrical
  • Parabolic
  • Paraboloidal
  • Elliptical
  • Toroidal
  • Prolate / Oblate Spheroidal
  • Bispherical
  • Conical
  • Bipolar

Rotation

Applies rotation to the rendered image.

  • 0 deg (normal)
  • 90, 180, 270 deg
  • Custom angle

Notes

Not all options apply to every fractal type. Behaviour may vary depending on the rendering mode (standard, perturbation, slope, or non-raster fractals).
