Paul's Fractal Generation Program

Paul's Fractal Generation Program is used for generating a select set of Fractals. This program is a Windows 64 bit version of a very old MS-DOS program, MANP, created to explore Fractals, especially the Mandelbrot Set which appeared in Scientific American August 1985. There is also a 32 bit version of ManpWIN.

ManpWIN brings together ideas found in a number of fractal programs, including the original program Manp. FRACTINT, TieraZon and original FORTRAN sources from Art-Matrix contribute heavily to the development of ManpWIN. More recent exploration of the Fibonacci Series and Fourier Analysis have been added for experimentation. Both FRACTINT and TiearaZon have helped with the development of filters and colour mapping.

Script files to create both zoom animations and Julia Set (fractals that require a “seed” value) animations have also been added. These create a series of still images in true colour PNG (Portable Network Graphics) format.

Paul's Fractal Generation Program is designed to support 24 bit colour video modes.

Arguments for parameter files (*.par) may be passed in to the program to allow linking with other programs or from batch files.

Both mouse and keyboard control is available.

Work is being undertaken at including Perturbation Theory for high speed at deep zooms, multi-threading and slope calculations.

The name MANPWIN is derived from the very old MS-DOS program MANP (Mandelbrots by Paul).
