Colour Menu

Colour Controls

QOpen palette colour map (*.map)
Loads FRACTINT-compatible palette files. These define how iteration values map to RGB colours using up to 256 entries.
LOpen true colour map (*.col)
True colour maps define RGB start values and increments (6 parameters total) used to generate smooth colour gradients.
NSave true colour map
Ctrl+NSave palette colour map
+ / -Colour cycling
Cycles the palette to create animated effects. Use + or - to change direction (may require pressing twice).
CChange colour scheme
Switches between palette and true colour modes, and allows generation of random true colour palettes.
Ctrl+SSet inside colour
Defines the colour used for points inside the fractal set.
Ctrl+EEdit palette
Provides tools for modifying palette colours across the full iteration range.
For advanced palette editing, see Palette Editing.
Ctrl+RGet Pixel Colour
Get the RGB colour of any pixel on the screen.

For more details on colour modes and behaviour, see More About Colour.
