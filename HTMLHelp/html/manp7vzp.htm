Fractal Menu

Overview

The Fractal Menu provides access to core functions for selecting, configuring, and navigating fractals.

Keyboard shortcuts are shown alongside each command.

Commands

T Fractal Type
Selects the fractal type and associated options such as subtype, power, and colour settings.
See Fractals Supported
Ctrl+T Restart Fractal with Current Parameters
Recalculates the fractal using the current parameter settings.
X Fractal Options
Controls plotting mode, threshold, log scaling, biomorph colouring, decomposition, filters, and colouring methods.
See Fractal Options
D 3D Parameters
Controls 3D rendering including scaling and rotation along all axes.
Tab Display Location
Displays the current location within the fractal, including coordinates, parameters, and precision information.
Ctrl+F Enter Fractal Co-ordinates
Allows manual entry of a specific location within the fractal.
F TieraZon Filter FD Options
Provides additional filtering options for supported TieraZon filters.
See Fractal Options
