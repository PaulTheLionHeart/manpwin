Fractals Menu

Fractal Type                                                              (key T)

This selects the particular fractal and any options such as sub-type, power or colour options.

Restart Fractal with Current Parameters              (Ctrl+T)

Fractal Options                                                          (key X)

This options selects plotting mode, threshold, log scale start value (0 for off), biomorph and decomposition. All the filter and colour methods are available in this menu item.

3D Parameters                                                           (key D)

When plotting as 3D, the following parameters: Scaling x, Scaling y, Scaling z, Rotation x (deg), Rotation y (deg), Rotation z (deg).

Display Location                                                       (key Tab)

Displays the location of the area within the fractal as well as parameter values and precision information.

Enter Fractal Co-ordinates                                      (Ctrl+F)

Enter the values for a particular location on the screen.

TieraZon Filter FD options                                      (key F)

Certain TieraZon filters allow a range of optional filtering methods to be used. Refer Fractal Options.
