This software is based in part on the work of the Stone Soup Group responsible for the development of FRACTINT. Many of the plotting methods and periodicity are taken straight from the FRACTINT source code.

Windows, XP, Vista and Windows 7 are trademarks of Microsoft Corporation.

The Special Newton fractals, Matein fractal and Rational Maps are based on FORTRAN code obtained from Art Matrix.

Some fractals are taken from the source code of the program TieraZon. The true colour palette process, several filter methods and colour methods were inspired by TieraZon. By: Stephen C. Ferguson http://home1.gte.net/itriazon/ email: itriazon@gte.net

The “Bignum Library” for arbitrary precision arithmetic was supplied by Wesley Loewer's Big Numbers. (C) 1994, Wesley B. Loewer.

Thanks to Marcus Rezende for his ideas and assistance in developing many new fractals and ideas.

Thomas Kim helped greatly with developing the arbitrary precision library MPFR used for deep zooming and Perturbation.

Many have helped with the development of Perturbation Theory for deep zooming. K.I. Martin gives a basic understanding of Perturbation Theory in SFT Maths PDF. Much of my early perturbation code was based on the work of Shirom Makkad https://github.com/ShiromMakkad/MandelbrotPerturbation. See also discussion on this topic in  Perturbation Code for Cubic and Higher Order Polynomials and Perturbation formula for the burning ship fractal.
