Acknowledgments

Foundations

This software is based in part on the work of the Stone Soup Group responsible for the development of FRACTINT. Many of the plotting methods and periodicity checking originate from the FRACTINT source code.

Windows, XP, Vista and Windows 7, 8, 10 and 11 are trademarks of Microsoft Corporation.

The Cubic, Special Newton fractals, Matein fractal and Rational Maps are based on FORTRAN code obtained from Art Matrix.

Fractal Contributions

Some fractals are derived from the program TieraZon. The true colour palette process, along with several filter and colour methods, were inspired by this work.
