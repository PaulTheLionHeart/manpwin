Fractal Options

This options selects plotting mode, basic parameters such as threshold, log scale start value, biomorph and decomposition. Filters from both FRACTINT and TieraZon can be selected as well as colour modes from TieraZon.

Basic parameters:

·       Threshold set the maximum test for the number of iterations for each pixel before continuing.

·       Teserral fill colour allows a different colour to fill the rectangles.

·       The log scale is useful for large values of threshold where the range of counts from the number set in the log value to threshold are compressed into a logarithmic range to fit 255 colours. 0 means no log scale.

·       Biomorph allows the fractal to have 'lifelike' properties. The value sets the biomorph colour.

·       Decomposition divides the phase of each section into colours. The number of colour divisions is set by the decomposition colour.

·       Orbit colour used in plotting orbits for Julia sets. Turn off if 0. Also used in Julia set animation.

·       Distance Estimation gives some unusual effects.

There are 8 plotting modes:

·       Solid Guessing (default) which is a multiple pass method and the fastest for most fractals. The number of passes can be specified in this dialogue.

·       Boundary Tracing which is quick for certain fractals but can lose detail for 'islands'.

·       Teserral Plotting where the screen is divided into rectangles and filled progressively.

·       Single pass which processes each pixel and does no guessing. Slow but precise.

·       Double pass which is similar to above but the first pass is 4 times faster for the first pass.

·       Horizontal and vertical blinds allow a quick view of a fractal.

·       Slope Forward Differencing. This forces a single pass as all pixels need to be calculated to get fractional slope values and is slow.

Filters and colour methods:

Different colours are used to display other characteristics of the fractal besides iteration count. Some beautiful effects can be created. A subset of these are available for Perturbation fractals.

Rotation angle:

The fractal may be rotated by 0, 90, 180 or 270 degrees.

Changing Co-ordinate System:

Many FRACTINT and ManpWIN fractals can be viewed in the following co-ordinate systems: Cartesian, Spherical, Cylindrical, Parabolic, Paraboloidal, Elliptical, Bipolar, Toroidal, Prolate Spheroidal, Oblate Spheroidal, Bispherical or Bipolar Cylindrical, effectively increasing the number of possible images created by a factor of 12. Cartesian is the default co-ordinate system. Not all fractals lend themselves to changes in co-ordinate system.
