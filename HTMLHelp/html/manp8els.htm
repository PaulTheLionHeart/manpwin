Fractals

Overview

The term fractal was coined by Benoit Mandelbrot in 1975, derived from the Latin fractus (broken) and frangere (to break). Fractals provide a way of visualising infinity, as their defining property is self-similarity — patterns that repeat at ever smaller scales as one zooms deeper into them.

One of the most fascinating aspects of fractals is that extremely simple mathematical rules can generate remarkably complex and beautiful structures. This principle has applications beyond visual exploration, including areas such as fractal compression.

Fractals and Nature

Fractals often resemble patterns found in nature. Structures such as coastlines, clouds, trees, and ferns exhibit fractal-like properties. Techniques such as Iterated Function Systems (IFS) can be used to synthesise highly realistic natural forms using relatively simple equations.

Complex Plane and Julia Sets

Many fractals are defined in the complex plane, using numbers of the form a + ib. A common pairing exists between two related forms:

  • Mandelbrot-type fractals — defined by varying the complex parameter c
  • Julia sets — generated using a fixed value of c (the seed point)

Each point in the Mandelbrot set corresponds to a unique Julia set. In ManpWIN, you can explore this relationship by toggling between a fractal and its corresponding Julia set using the Space bar. The Julia set is generated using the point at the centre of the current image.

Fractals and Waveforms

Many waveforms can also be viewed through a fractal lens. In theory, a waveform contains an infinite number of harmonics. ManpWIN allows exploration of both the time domain and the frequency domain, revealing the underlying structure of these signals.

Notes

Fractals are not limited to a single definition or field. They sit at the intersection of mathematics, art, and nature — offering both a tool for analysis and a source of visual beauty.
